Space Cat Shooter Game

In the Space Cat Game, you control a space-traveling cat navigating through an endless space filled with obstacles and collectibles. The primary objective is to survive as long as possible while increasing your score by collecting items and avoiding hazards.

The game features a simple yet engaging gameplay loop, where players can earn points by collecting items and avoiding obstacles. The game is designed to be easy to pick up and play, making it suitable for players of all ages.

The game is built using Python and Visual Studio Code, showcasing my skills in game development and programming. The artwork was made by me using Clip Studio Paint.

The game is inspired by my cat Johan, who is a playful and curious feline. The game captures his adventurous spirit as he explores the vastness of space, destroying objects and meowing playfully.

Memory Card Game

The Memory Card Game is a fun and interactive game designed to test and improve your memory skills. The objective of the game is to match pairs of cards by flipping them over two at a time. If the cards match, they remain face up; if not, they are flipped back over.

The game features three difficulty levels: easy, medium, and hard. Each level offers a different number of cards to match. The easy level has 4 cards, the medium level has 16 cards, and the hard level has 36 cards.

The game features a timer that increases with the difficulty level,and a bonus score for correct cards streaks.

The game is built using c# and Windows Forms App (.NET Framework).

Calculation Game

The Calculation Game is an interactive game designed to test and improve your math skills. The objective of the game is to solve a series of math problems, there is also a timer that calculates the time taken to solve the problems, that is later added into your score. So the lower your score is, the better you are at math!

The game is built using c# and Windows Forms App (.NET Framework).
